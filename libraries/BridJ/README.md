# BridJ

Project home:
http://bridj.googlecode.com/

# Getting Started

* Usage: see `Examples/BasicExample`
* Build from sources:
  ```
  git clone http://github.com/ochafik/nativelibs4java.git
  cd nativelibs4java/libraries/BridJ
  mvn clean install
  ```
